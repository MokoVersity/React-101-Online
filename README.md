本課程是[React 101 實戰教學：開始用 React 打造 Web Frontend](https://www.mokoversity.com/training/React-101)的線上課程版本。本課程將總覽學習 React 的入門知識，目標是為初學者，製作優質的 React 的入門級課程。

## 課程規劃

* 採取主題式短片製作，總計 30 部影片，共 120 分鐘的線上影片
* 課程影片為特別製作，非現場錄影重製
* 提供線上聊天室討論
